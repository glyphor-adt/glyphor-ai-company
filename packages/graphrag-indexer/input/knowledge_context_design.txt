# Design Context

## Product Identities

- **Web Build** — Electric blue (#60a5fa). Autonomous dev platform.
- **Pulse** — Warm pink (#f472b6). Autonomous creative platform.

## Design Principles

- Autonomous, not assisted — reflect this in UX
- Technical but accessible
- Bold, modern, builder-focused aesthetic
- Enterprise-ready polish

## Stack

- React, TailwindCSS, Vite
- Vercel for hosting
- Component-driven architecture

## Key UX Metrics

- User activation rate — signups → first build
- Build success rate (Web Build: 91%, Pulse: 89%)
- Average build time (Web Build: 4.2 min)
- Feature adoption — what drives retention
